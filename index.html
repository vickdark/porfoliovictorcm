Sistema de Transporte (PHP)

Sistema de gestión de transporte con manejo de usuarios, roles, órdenes y reportes.

Problema: La operación se gestionaba completamente de forma manual, sin herramientas centralizadas para el control de datos, lo que generaba inconsistencias y dificultaba la trazabilidad.

Solución: Desarrollo de un sistema integral de gestión de transporte con arquitectura MVC, implementando módulos de usuarios, roles, órdenes y reportes. Se realizó ingeniería inversa de sistemas dispersos para centralizar la información, generando formatos únicos para establecimientos comerciales, checklists preoperacionales y un centro de KPIs con estadísticas generales del sistema.

Gestión ITAM

Sistema administrativo para gestión de procesos internos de infraestructura tecnológica y mesa de ayuda

Problema: La gestión de infraestructura tecnológica y mesa de ayuda se llevaba en documentos físicos, dificultando el control de hojas de vida, cambios programados, mantenimientos y auditorías. Además, existían problemas recurrentes para obtener firmas en formatos y atender solicitudes de manera oportuna.

Solución: Sistema administrativo integral que digitaliza la gestión de activos tecnológicos, implementando control de ciclo de vida, programación de mantenimientos y captura digital de firmas. Se integró una mesa de ayuda para gestionar solicitudes de mantenimiento, garantizando trazabilidad completa y cumplimiento de estándares de auditoría.

Sistema de Transporte Conductores (Ionic)

Aplicación móvil para conductores, integrada con sistema backend.

Problema: En el sistema de transporte, los conductores solo tenían acceso limitado y debían reportar manualmente el kilometraje inicial, final y los viajes realizados, generando demoras e inconsistencias.

Solución: Aplicación móvil desarrollada en Ionic que automatiza el envío de kilometraje y reportes de viajes. Implementa seguimiento GPS en tiempo real, integrado con un panel de tracking web que permite al cliente monitorear toda la operación de conductores en un mapa interactivo, mejorando el control y la eficiencia operativa.
