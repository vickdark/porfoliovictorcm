# Portfolio de Desarrollador PHP

## Visión General

Crear un portfolio web profesional, limpio y directo para un desarrollador PHP. El sitio debe mostrar de forma clara y atractiva sus habilidades, proyectos y experiencia, con un enfoque en el desarrollo backend y enlaces directos a sus repositorios.

## Estructura y Diseño

*   **Diseño:** Tema oscuro para el cuerpo de la página con alto contraste. Las tarjetas de proyecto son transparentes con un borde sutil, integrándose perfectamente en el fondo.
*   **Paleta de Colores:**
    *   **Fondo principal:** Negro/Gris oscuro (`#121212`).
    *   **Tarjetas de proyecto:** Transparentes con borde claro.
    *   **Texto en tarjetas:** Blanco/Gris claro (`#EAEAEA`, `#ccc`).
    *   **Botones de acción:** Azul primario de Bootstrap (`#0d6efd`).
    *   **Color de acento:** Lavanda (`#c77dff`) para el subtítulo del encabezado.
*   **Responsivo:** Totalmente adaptable a dispositivos móviles y de escritorio.
*   **Tecnologías:** HTML5, CSS3, Bootstrap 5, JavaScript (para animaciones de scroll), Devicon.

## Plan de Implementación y Características

1.  **Encabezado:**
    *   Nombre y título con tipografía grande y clara.
    *   Imagen de fondo relevante con una superposición de gradiente oscuro para asegurar la legibilidad del texto.
2.  **Sobre mí:**
    *   Sección con dos columnas que detallan el enfoque en backend y la experiencia en calidad y soporte.
3.  **Habilidades:**
    *   Grid de iconos representando las tecnologías del desarrollador.
    *   Diseño oscuro y minimalista, consistente con el resto del cuerpo de la página.
4.  **Proyectos Destacados:**
    *   **Diseño:** Un grid de tarjetas transparentes con bordes redondeados y un sutil efecto de sombra.
    *   **Contenido:** Cada tarjeta contiene:
        *   Título del proyecto.
        *   Descripción concisa.
        *   **Iconos de Tecnología:** Se utilizan iconos de Devicon con sus colores de marca originales. Para tecnologías sin icono (como MVC), se usa una píldora de texto con fondo oscuro.
        *   **Botón de Llamada a la Acción:** Un botón azul sólido y prominente. Para repositorios públicos, el texto es "Ver Repositorio". Para el proyecto privado, el texto es "Ver Detalles" y el botón está visualmente desactivado.
5.  **Contacto:**
    *   Iconos grandes y claros para enlazar a GitHub, LinkedIn y correo electrónico.
